# 知识点

## 曲线

- 曲线一般与 Animation Controller 连用
- 创建一个状态机后，将动作拖入
- 添加一个与 曲线 名字相同的参数
- 将 状态机 拖入模型中
- 之后需要通过曲线来改变参数的变化
- **但是实际开发中使用较少**

## 动画事件

- **使用次数极多
- 必须确保附带的脚本里面有同名函数

## 遮罩

- 使用较多
