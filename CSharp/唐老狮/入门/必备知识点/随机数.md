# 知识点

### 创建随机数对象

- 固定写法
	- `Random 随机数变量名 = new Random();`

### 生成随机数

 - 左包含，右不包含
 - `int i = 随机数变量名.Next();`生成一个非负数的随机数
	 - `.Next(100)` 0~99
	 - `.Next(5,100);` 5~99
