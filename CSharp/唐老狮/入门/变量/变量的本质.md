# 知识点

## 变量的存储空间

- 变量的存储空间
	- 可以通过 `sizeof` 获取变量类型所占的内存空间
	- 1 byte = 8bit ；KB MB GB T 都是 1024 
	- 字节数
		- 字节数越小，范围越小
		- uint / int 4 
		- ushort / short 2
		- ulong / long 8
		- sbyte 1
		- float 4
		- double 8 
		- bool 1
		- char 2
		- string 无法被 `sizeof` 获取，因为其可变
## 变量的本质
- 变量的本质
	- 本质
		- 二进制 0 和 1
	- 二进制与十进制转换
