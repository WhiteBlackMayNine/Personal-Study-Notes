## Text Elements
跳跃系统 ^za0sgvMd

持续按住空格 ——> 玩家上升
一旦松开空格 ——> 玩家下落
同时 除非玩家上升到高度上限
否则玩家下落速度提高 ^bTpgX32C

同时 还有二段跳 ^J4mzo4TC

设计思路 ^tqjA2KAW

当玩家处于地面时，点击空格则设置 IsCanJump 为 True
然后 当IsCanJump 为true 时按住空格 进行计时并修改速度
当时间大于0.5f将IsCanJump设置为 false 并IsJumpTop 设置为 true
当空格抬起时 修改玩家所受的重力系数
当玩家落地时，重置所有条件 ^cbd7GDPk

二段跳需要注意一下代码顺序，必须在跳跃之前就检测
否则，当键盘抬起时，IsSecondJump 为 true，且其他条件也为 true
导致 ——> 松开空格时便进行二段跳
所以 建议将逻辑放到 跳跃之前 ^UdEX4nQ3
