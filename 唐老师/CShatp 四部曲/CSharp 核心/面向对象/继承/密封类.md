# 知识点

## 基本概念

### 概念

- 使用密封关键字`sealed`修饰的类
### 作用

- 无法被继承
## 实例

- `class sealed Father{ } `
- `class sealed Son : Father{ }`
## 作用

- 在面向对象程序的设计中，密封类的主要作用就是不允许最底层子类被继承
- 可以保证程序的规范性、安全性
