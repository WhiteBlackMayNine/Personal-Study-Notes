# 知识点

## Model 模型页签是设置什么的

- 该页签主要是用于设置
- 比如
	- 模型比例设置
	- 是否导入模型中的摄像机和光源
	- 网格压缩方式 等等相关信息
- 修改模型中存储的各种元素和属性
- 最终会影响在 Unity 中使用模型时的一些表现
