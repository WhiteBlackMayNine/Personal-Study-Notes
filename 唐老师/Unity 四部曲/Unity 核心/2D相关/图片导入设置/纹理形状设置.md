# 纹理形状主要是设置什么

- 纹理不仅可以用于模型贴图
- 还可以用于制作天空盒和反射探针
- 纹理形状设置 主要就是用于在两种模式之间进行切换
# 参数讲解
## Texture Shape

- 纹理形状
### 2D

- 2D纹理，最常用设置，这些纹理将使用到模型和GUI元素上
- 图片就是单纯的铺在模型上或者UI上
### Cube

- 立方体贴图，主要用于天空盒和反射探针
