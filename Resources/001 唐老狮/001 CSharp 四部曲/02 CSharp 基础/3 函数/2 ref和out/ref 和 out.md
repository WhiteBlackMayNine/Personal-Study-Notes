# 知识点

## 学习它的原因

- 可以解决在函数内部改变外部传入的内容，里面变了外面也变
## 使用

- `ref`
	- 作用
		- 函数参数的修饰符
		- 当传入的值类型在内部修改时，或引用类型参数在内部重新声明时
		- 外部的值都会改变
	- 使用方法
		- 传入参数时在前面直接加上
		- `ref int a,……`
- `out`
	- 作用
		- 
	- 使用方法
		- 与 `ref` 相同
## 区别

- `ref` 传入的变量必须初始化（在函数外部），但`out`不用
- `out`传入的变量必须在内部赋值，但`ref`不用
- 都是为了安全
