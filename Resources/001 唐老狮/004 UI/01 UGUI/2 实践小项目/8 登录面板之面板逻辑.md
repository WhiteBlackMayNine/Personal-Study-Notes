# 明确需求

- 点击 注册
	- 来到注册面板
- 点击 确定
	- 验证账号密码是否正确
- 记住密码、自动登录也有相关逻辑
	- 但部分逻辑需要等到以后才能做

- 所有，总共有两部分需求
	- 数据相关
	- 逻辑相关
# LoginData

- 数据结构类
	- 用于存储数据
# LoginMgr

- 登录管理器
	- 主要用于管理登录相关数据
		- 登录数据
		- 注册数据
		- 服务器数据
	- 提供存储获取相关方法属性
	- 检测账号密码
	- 改变登录服务器ID
# LoginPanel

- 获取面板上的组件
- 为其添加相应的逻辑

- 点击注册、登录按钮
	- 前者跳转到 注册面板
	- 后者则去验证账号密码是否正确
- 账号、密码 文本文本输入控件
	- 输入账号密码
- 记住密码、自动登录 Toggle 多选框
	- 根据两者的勾选情况，去实现相应逻辑

- LoginData 会保存 登录面板的信息
- LoginMgr 则会去加载 LoginData 保存的信息
- 但这些功能得到后面才能实现
