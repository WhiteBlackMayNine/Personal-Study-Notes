# 3D模式何时出现

- 当Canvas的渲染模式设置为 **世界空间3D渲染模式** 时
	- ——> （ World Speac）

- 这时 Canvas Scaler 的缩放模式会强制变为 World （3D世界模式）
# 参数讲解

- Dynamic Pixels Per Unit
	- Ul中动态创建的位图（例如文本）中，单位像素数（类似密度）
	- **这个值越大，越清晰**
- Reference Pixels Per Unit
	- 单位参考像素，多少像素对应 Unity 中的一个单位（默认一个单位为100像素）
# 总结

- 3D模式
	- 只有在3D渲染模式下才会启用的模式
	- 主要用于控制该模式下的像素密度

- 该模式了解即可，一般不修改
