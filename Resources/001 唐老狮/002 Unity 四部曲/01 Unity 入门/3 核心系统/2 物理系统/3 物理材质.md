# 知识点

## 创建物理材质

- 项目 —— 点击 【+】 —— 物理材质
## 物理材质参数

### 动态摩擦力

- 已在移动时使用的摩擦力
- 通常在 0~1 之间
###  静态摩擦力

- 当对象静止在表面上使用的摩擦力
- 通常在 0~1 之间
### 弹力

- 表面的弹性
- 值为 0 时不反弹，值为 1 时反弹不产生能力损失
### 摩擦组合

- 两个碰撞物体的摩擦力的组合方式
### 反弹合并

- 两个碰撞对象的弹性的组合方式
- 其模式 与Driction Combine 模式相同
