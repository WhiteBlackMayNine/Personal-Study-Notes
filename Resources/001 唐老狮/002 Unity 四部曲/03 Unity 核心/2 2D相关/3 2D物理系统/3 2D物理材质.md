# 什么是物理材质

- 物理材质是用于决定在物体产生碰撞时这些物体之间的摩擦和弹性表现的
- 通过物理材质我们可以做出类似 斜坡不滑落，小球反弹等效果
# 参数讲解
## Friction

- 摩擦系数，为0表示没有摩擦力
## Bounciness

- 反弹程度。0表示没有弹力，1表示完美弹力（没有能量损失直弹）
