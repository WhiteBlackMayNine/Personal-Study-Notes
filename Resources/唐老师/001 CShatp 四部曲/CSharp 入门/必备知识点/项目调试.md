# 知识点

### 断点调试

- 排除代码错误 / 理解程序逻辑
	- 断点调试
		- 创建
			- 在VS左侧条上点一次（或者按F9）
		- 作用
			- 程序运行到断点处会中断运行
			- 可以使用VS上方的一系列 按钮/快捷键来进行操作（如 F10）
