# 知识点

## 基本概念

- 数组的数组（一维数组里元素是数组），每个维度的数量不同（每行的列数可能不同）
## 声明

- `变量类型[][] 数组名;`
- `变量类型[][] 数组名 = new 变量类型[行数][];`
- `变量类型[][] 数组名 = new 变量类型[行数][]{ 一维数组1,一维数组2,…… };`
	- `int arr3 = new int[3][]{new int[]{1,2,3 },new int[]{ 1,2 },new int[]{ 1 }};
- `变量类型[][] 数组名 = new 变量类型[][]{ 一维数组1,一维数组2,…… };`
- `变量类型[][] 数组名 = { 一维数组1,一维数组2,…… };`
## 使用

- 长度
	- 行数
		- `交错数组名.GetLength(0)`
	- 列数
		- `交错数组名[行数].Length`
			- 得到某一行的列数
- 获取元素
	- 不能越界
- 修改元素
	- 直接赋值
- 遍历
	- 根据长度进行循环语句遍历
