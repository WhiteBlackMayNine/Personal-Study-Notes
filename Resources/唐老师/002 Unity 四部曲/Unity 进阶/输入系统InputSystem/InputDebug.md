# 什么是Input Debug

- InputDebug 顾名思义是输入调试器的意思
- 我们可以通过输入调试窗口检测输入相关信息
- 当我们的 输入不按预期工作 时，可以通过它来排查问题
# 打开 InputDebug 窗

- Window（窗囗）——> Analysis（分析）——> Input Debugger（输入调试器）
- PlayerInput组件 ——> 0pen Input Debugger
