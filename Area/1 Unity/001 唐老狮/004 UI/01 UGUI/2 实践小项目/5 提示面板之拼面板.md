# 拼面板

- 根据示意图
- 进行拼接
# 注意

- 在拼界面时
	- Panel 的 Image 并不需要
		- 可以移除
	- 可以把示意图放在最下面，然后把透明度调低
		- 在拼界面时，可以对照着拼
	- 如果图片上有坑坑洼洼的地方
		- 建议在九宫格选定时，避开那些地方
	- 给 Text 添加组件 OutLine
		- 可以给字体增加描边效果
	- Button
		- 可以把下面的 Text 删掉
	- 注意
		- Text 为旧的UI文件组件，可以使用Windows自带的字体
		- 但富文本不可以
