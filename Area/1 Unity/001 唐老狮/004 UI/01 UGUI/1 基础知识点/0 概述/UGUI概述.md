# UGUI是什么

- UGUI是Unity引擎内自带的UI系统
- 官方称之为：Unity Ul
- 是目前Unity商业游戏开发中使用最广泛的U系统开发解决方案
- 它是基于Unity游戏对象的UI系统，只能用来做游戏UI功能
- 不能用于开发Unity编辑器中内置的用户界面
# UI系统的重要性

- UI是游戏中最重要的部分
- 不管是2D游戏还是3D游戏
- 几乎所有游戏功能都和UI有关系
- 所以它是极其重要的知识点
