# 知识点

## 基本概念

- 概念
	- 用密封关键字修饰的重写函数
- 作用
	- 让虚方法或抽象方法之后不能再被重写
- 特点
	- 和`override`**一起出现**
## 实例

- `public sealed override void Speak(){ }`
