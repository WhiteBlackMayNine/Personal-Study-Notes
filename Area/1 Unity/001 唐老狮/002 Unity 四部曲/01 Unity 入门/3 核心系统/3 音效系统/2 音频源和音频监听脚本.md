# 添加 音效原脚本

- 如果想要把导入的音频在游戏里面去播放的话
- 就需要使用到 音频原脚本
- ——> 选择一个物体，点击 Add Component 
- ——> 选择 Audio Source
# 参数详细讲解
## AudioClip

- 如果不去关联一个音频文件，那么对象是不会去播放的

- 关联希望播放的音效文件的，想要播放谁
- 就可以把这个音效文件拖入到这个参数里
## Output

- 输出的一个设置
- 除非是做一些有关音乐的游戏
- 否则一般都不会去设置这个参数
## Play On Awake

- 对象创建时就播放音乐
- **也就是开关启动游戏就播放**
# 音频监听脚本 AudioListener

- 默认挂载到场景上的主摄像机
- 如果没有这个脚本，我们是听不到音效的
- 可以理解为一个收声的“耳朵”
